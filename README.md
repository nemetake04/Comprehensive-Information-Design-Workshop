# Comprehensive-Information-Design-Workshop
学部3年　情報システムデザイン総合演習　第四班「体感型絵本-ファイブック-」

🌱 プロジェクト概要  
本プロジェクトは、春・夏・秋・冬の四季に合わせて色や光が変化する「体感型絵本」 を制作したものです。
紙の仕掛け、光の演出、キャラクターデザイン、立体的な本の構造を組み合わせ、
「めくる」「回す」「差し込む」といったアクションがそのまま季節の変化になる読書体験を目指しました。

🎯 目的  
・本というアナログ媒体に“体験”と“驚き”を追加する  
・子どもでも大人でも楽しめるインタラクティブ作品をつくる  
・紙工作 × 光学演出 × 映像記録による総合的なアート表現を試みる  

🍀 コンセプト

《季節をめくる。光が語る。》

本の中を覗くと、小さな部屋のような空間にキャラクターが配置されており、
仕掛けを動かすことで色が変化し、物語が進んでいく仕様です。

📚 制作物（4冊の季節絵本）
それぞれ独自のギミックを持った4冊を制作しました。

🌸 春 – 回転ギミック（緑→ピンク）
桜イメージ。回すと季節の訪れを感じさせる色に。

🌊 夏 – 回転＋差し込み（緑→青）
海のような青が広がる。差し込み操作で変化。

🍁 秋 – 回転ギミック（緑→黄→赤）
紅葉の流れを色の変化で再現。

❄ 冬 – カーテン操作（白ベース）
雪景色をモチーフに、カーテンを開く/閉じることで印象が変化。

🛠 技術・構造  
・ハードカバー風の装丁（本型小物入れを改造） 
・テープLEDによる発光ギミック  
・透過紙やカラーセロファンで色変化を制作  
・紙細工で内部の小部屋を構成  
・映像撮影による展示記録  

🎬 展示について

教室一室を貸し切り、実際に数十名の方に体験していただきました。
